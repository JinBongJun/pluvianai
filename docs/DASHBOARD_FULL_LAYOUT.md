## 주요 섹션 상세 설명

### 1. 상단 헤더 바 (고정, 모든 페이지 공통)

```
┌─────────────────────────────────────────────────────────────────────┐
│  Projects        [🔍] [🔔 3] [👤 User Name ▼] [+ New Project]      │
└─────────────────────────────────────────────────────────────────────┘
```

**구성**:
- **좌측**: 페이지 제목 ("Projects")
- **우측**:
  1. `🔍` 검색 아이콘 → 전역 검색 모달 (Cmd/Ctrl+K)
  2. `🔔 3` 알림 아이콘 → 알림 센터 (읽지 않은 알림 3개)
  3. `👤 User Name ▼` 유저 프로필 → 드롭다운 메뉴:
     - Usage
     - Settings
     - Billing
     - Logout
  4. `+ New Project` 버튼 → 프로젝트 생성 모달

---

### 2. 검색/필터/정렬 영역

```
┌─────────────────────────────────────────────────────────────────────┐
│  🔍 Search projects...    [Filter ▼] [Sort ▼] [Grid] [List]        │
│                                                                     │
│  5 Projects    Sort: Recent Activity ▼    Views: [Grid] [List]     │
└─────────────────────────────────────────────────────────────────────┘
```

**기능**:
- **검색창**: 프로젝트 이름/설명 실시간 필터링
- **Filter 버튼**: 고급 필터 패널 (상태, 플랜, 날짜 등)
- **Sort 드롭다운**: Recent Activity, Name A-Z, Most Calls, Most Alerts
- **View 전환**: Grid 뷰 / List 뷰

### 4. 프로젝트 카드 그리드

```
┌─────────────────────────────────────┐
│  [아이콘]  My Project    [owner]   │
│                                     │
│  Description of the project goes    │
│  here and can be multiple lines...  │
│                                     │
│  ─────────────────────────────────  │
│  📊 1,234  ⚠️ 2  🕐 2h ago         │
│                                     │
│                              [→]    │
└─────────────────────────────────────┘
```

**상태 인디케이터**:
- `📊 1,234`: 24시간 호출 수
- `⚠️ 2`: 알림 수 (색상으로 위험도 표시)
  - 0개: `✅` 또는 표시 안 함
  - 1-2개: `⚠️` (주황색)
  - 3개 이상: `🔴` (빨간색)
- `🕐 2h ago`: 마지막 업데이트 시간
- `[→]`: 클릭 가능 표시 (Railway/Supabase 스타일)

**인터랙션**:
- 전체 카드 클릭 → `/dashboard/{projectId}` (프로젝트 상세 Overview)

---

### 5. 사이드바 하단 유저 정보 (보조 역할)

```
┌─────────────────────┐
│  [User Info]        │
│  👤 User Name       │
│  user@example.com   │
│  [Free] 배지        │
│                     │
│  Settings           │
│  Logout             │
└─────────────────────┘
```

**역할**:
- 상단 헤더의 유저 프로필이 **메인 인터랙션 포인트**
- 사이드바 하단은 **보조 정보** (모바일/작은 화면에서 유용)

## 레이아웃 특징 요약

### 공간 배치
- **좌측 사이드바**: 고정 너비 (256px), 네비게이션 + 프로젝트 리스트
- **상단 헤더 바**: 고정 높이, 모든 페이지 공통
- **메인 콘텐츠**: 
  - 좌측: Usage/Alerts 섹션 (고정 또는 스크롤 가능)
  - 우측: 프로젝트 그리드 (스크롤 가능)

### 색상/스타일
- 배경: `#000314` (Deep Navy)
- 카드: Glassmorphism (반투명, 블러 효과)
- 강조 색상: Purple, Cyan, Red (Glow 효과)
- 폰트: Inter (본문), JetBrains Mono (숫자)

### 반응형 고려
- 데스크탑: 3-컬럼 그리드 (사이드바 + Usage/Alerts + Projects)
- 태블릿: 2-컬럼 (사이드바 접기 + 메인 콘텐츠)
- 모바일: 1-컬럼 (사이드바 햄버거 메뉴)

## 사용자 플로우 예시

### 시나리오 1: 문제 프로젝트 확인
1. `/dashboard` 진입
2. 좌측 Alerts 섹션에서 "Project 1: 2 alerts" 확인
3. Alerts 섹션의 "Project 1" 클릭
4. → `/dashboard/1/alerts`로 이동

### 시나리오 2: 새 프로젝트 생성
1. 상단 헤더의 `+ New Project` 클릭
2. 모달에서 프로젝트 이름/설명 입력
3. Create 버튼 클릭
4. → 새 프로젝트 생성 후 `/dashboard/{newId}`로 이동

### 시나리오 3: 프로젝트 검색
1. 상단 검색창에 "API" 입력
2. 실시간으로 "API Monitor" 프로젝트만 필터링
3. 프로젝트 카드 클릭
4. → `/dashboard/{id}`로 이동
