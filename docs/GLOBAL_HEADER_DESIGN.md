# AgentGuard 전역 헤더 설계

## 목표
- 모든 페이지에서 일관된 상단 헤더
- 표준 SaaS 패턴 적용 (Supabase, Vercel 등 참고)
- 사용자 친화적인 네비게이션

---

## 1. 헤더 구조

### 1-1. 기본 레이아웃

```
┌──────────────────────────────────────────────────────────────────────────┐
│ [AG Logo] / [Breadcrumb]  [Feedback] [🔍 Search ⌘K] [❓] [💡] [👤]      │
└──────────────────────────────────────────────────────────────────────────┘
```

### 1-2. 요소별 설명

#### 좌측 영역
- **AG Logo**: AgentGuard 로고 (홈/대시보드로 이동)
- **Breadcrumb**: 현재 페이지 경로
  - `/ Organizations`
  - `/ Organizations / New`
  - `/ Organizations / {orgName}`

#### 우측 영역 (왼쪽부터)

1. **Feedback** (선택사항)
   - 사용자 피드백 수집
   - 버튼 또는 링크

2. **🔍 Search ⌘K**
   - 전역 검색
   - 키보드 단축키: ⌘K (Mac) 또는 Ctrl+K (Windows)
   - 검색 모달 열기

3. **❓ Help**
   - 도움말 드롭다운
   - 문서, FAQ, 지원

4. **💡 Suggestions** (선택사항)
   - 제안/아이디어
   - 피드백과 유사하지만 더 구체적

5. **👤 Profile**
   - 사용자 프로필 드롭다운
   - 메뉴 항목:
     - Usage (사용량)
     - Settings (설정)
     - Billing (결제)
     - Logout (로그아웃)

## 3. Org 스위처 (Org 컨텍스트 내)

### 3-1. 드롭다운 구조

```
┌──────────────────────────────────────────────────────────────────────────┐
│ [AG Logo] / {orgName} [FREE ▼]  ← 클릭 시 드롭다운 열림                │
└──────────────────────────────────────────────────────────────────────────┘
```

**드롭다운 메뉴**:
```
┌─────────────────────────────────────┐
│ 🔍 Find organization...             │
│                                      │
│ ✓ JinBongJun's Org                  │
│   My Company Org                     │
│   ─────────────────────────────────  │
│   All Organizations                  │
│   + New organization                 │
└─────────────────────────────────────┘
```

### 3-2. 인터랙션
- Org 선택 → 해당 Org로 전환
- "All Organizations" → `/organizations`로 이동
- "+ New organization" → `/organizations/new`로 이동

## 4. 스타일 가이드

### 4-1. 다크 테마 (기본)
- 배경: 다크 그레이 (#1a1a1a 또는 유사)
- 텍스트: 흰색/밝은 회색
- 버튼: 초록색 Primary 버튼

### 4-2. 레이아웃
- 상단 고정 (sticky header)
- 높이: 64px (일반적)
- 패딩: 좌우 24px

### 4-3. 반응형
- 모바일: 일부 요소 숨김 (Feedback, Suggestions)
- 태블릿: 모든 요소 표시
- 데스크톱: 전체 표시

---

## 5. 구현 우선순위

### Phase 1: 기본 구조
1. ✅ 전역 헤더 컴포넌트 생성
2. ✅ Breadcrumb 컴포넌트
3. ✅ 프로필 드롭다운

### Phase 2: 기능 추가
1. ✅ 전역 검색 (⌘K)
2. ✅ Help 드롭다운
3. ✅ Feedback 버튼

### Phase 3: 고급 기능
1. ✅ Org 스위처 (Org 컨텍스트 내)
2. ✅ 알림 배지 (선택사항)
3. ✅ 키보드 단축키

---

## 6. 참고 서비스

- **Supabase**: Breadcrumb + 검색 + 프로필
- **Vercel**: 로고 + 검색 + 알림 + 프로필
- **Railway**: 로고 + Help + 프로필

AgentGuard는 이들의 장점을 결합하여 일관된 헤더를 제공합니다.
