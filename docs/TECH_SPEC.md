# Tech Spec: 행동 분석(PostHog)

## 목표
- 프론트엔드 행동 분석을 PostHog로 수집한다.
- 페이지 뷰와 핵심 사용자 이벤트를 기록한다.
- 환경 변수로 손쉽게 on/off 가능하게 한다.

## 비목표
- 서버 사이드 이벤트 파이프라인 구축
- 복잡한 세그먼트/퍼널 대시보드 구성

## 요구사항
1) PostHog JS 초기화 및 Provider 구성
2) 페이지 뷰 자동 캡처
3) 환경 변수 미설정 시 비활성화
4) 주요 이벤트 확장 가능 구조

## 성공 기준
- PostHog 키가 설정되면 페이지 뷰가 수집된다.
- 키 미설정 시 에러 없이 렌더링된다.

## 리스크/가정
- 광고 차단기로 인해 일부 이벤트 누락 가능
