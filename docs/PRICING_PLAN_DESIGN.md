# AgentGuard 플랜 가격 및 기능 설계

## 📊 현재 상태 분석

### AgentGuard 핵심 기능 (실제 구현 기준)

**모니터링 & 추적:**
- ✅ API 호출 추적 (모든 LLM API)
- ✅ 비용 분석 및 계산
- ✅ 지연시간 분석
- ✅ 에러 모니터링

**품질 & 드리프트:**
- ✅ 기본 드리프트 감지 (길이, 구조)
- ✅ 향상된 드리프트 감지 (의미, 톤)
- ✅ 기본 품질 평가 (JSON/구조 검증)
- ✅ 고급 품질 평가 (LLM 기반 평가)

**분석 & 비교:**
- ✅ 모델 비교 (Benchmark)
- ✅ 에이전트 체인 프로파일링
- ✅ 비용 이상 감지
- ✅ 모델 최적화 추천

**협업 & 알림:**
- ✅ 팀 멤버 관리
- ✅ 알림 (Email, Slack, Discord)
- ✅ 리포트 생성 (수동/자동)
- ✅ 웹훅 통합

**고급 기능:**
- ✅ RBAC (역할 기반 접근 제어)
- ✅ 데이터 보관 (7일 ~ 365일)
- ✅ Self-hosted (Enterprise)

### 현재 플랜 문제점

1. **5개 플랜 → 너무 많음** (업계 표준: 3-4개)
2. **Indie ($19)와 Startup ($59) 차이가 큼** (중간 단계 필요)
3. **Free 플랜이 너무 관대함** (3 projects, 1K calls)
4. **가격대가 경쟁사 대비 높을 수 있음**

## 🎯 핵심 전략

### 1. Free 플랜 제한 강화
- **이유**: 현재 3 projects, 1K calls는 너무 관대함
- **변경**: 1 org, 1 project, 5K calls로 제한
- **효과**: 실제 사용 시 빠르게 Pro로 업그레이드 유도

### 2. Pro 플랜을 "Most Popular"로 강조
- **가격**: $49/month (기존 Startup $59보다 저렴)
- **기능**: 핵심 기능 모두 포함
- **마케팅**: UI에서 "Most Popular" 배지, 기본 추천

### 3. Enterprise 가격 인하
- **기존**: $499/month
- **새**: $299/month
- **이유**: 경쟁력 강화, Self-hosted 옵션으로 차별화

### 4. 연간 결제 할인
- **Pro**: 15% 할인 ($499/year)
- **Enterprise**: 20% 할인 ($2,870/year)
- **효과**: 연간 수익 예측 가능, 캐시플로우 개선

## 📝 참고사항

### Org 단위 과금
- 모든 제한은 **Org 단위**로 적용
- 예: Pro 플랜 = 3개 Org, 각 Org당 10 projects, 100K calls

### 소프트 리밋
- Enterprise의 "무제한"은 실제로는 소프트 리밋
- 예: API Calls 1M/월 초과 시 연락 요청

### 기능 차별화
- Free: 문제 발견만 가능 (해결 불가)
- Pro: 모든 핵심 기능 (대부분의 팀이 충분)
- Enterprise: 고급 기능 + Self-hosted (대기업 전용)
